Portfolio Artifacts

Artifact 1: DALL·E – Teaching AI to Imagine

A timeline and case analysis of DALL·E's evolution, applications, and limitations. Includes real-world examples, ethical concerns, and predictions for AI-driven creativity tools.

  • Tools: PowerPoint, AI image prompts, research documentation
  • Value: Demonstrates critical thinking, tech literacy, and awareness of generative AI’s impact on industries

Artifact 2: Body Fat Prediction using Machine Learning

A hands-on project applying supervised learning to predict body fat percentage using linear regression. This project includes data cleaning, outlier detection, and performance evaluation using a Body Fat Prediction Dataset downloaded from Kaggle.

  • Tools: Python, Jupyter Notebook, Pandas, Matplotlib, Scikit-learn
  • Value: Demonstrates end-to-end data science workflow including EDA, data wrangling, and regression modeling.

Artifact 3: Neural Networks as Pizza – Teaching AI Concepts

A playful but insightful analogy-driven explanation of neural networks using the process of making pizza. This artifact was developed for a classroom discussion and demonstrates how storytelling and analogy can make complex AI concepts accessible to non-technical audiences.

  • Tools: ChatGPT, analogy-based instruction, instructional design thinking
  • Value: Highlights technical communication skills and the ability to teach AI through creative, simplified language

Artifact 4: Student Success Prediction – Exploring Bias in Decision Trees

A classification project using student test scores and background data to explore model bias in predicting academic success. Highlights the impact of class imbalance and demographic variables on decision tree outcomes.

  • Tools: Python, Jupyter Notebook, Pandas, Seaborn, Scikit-learn
  • Value: Shows how human bias in data can affect AI fairness and decision-making.
