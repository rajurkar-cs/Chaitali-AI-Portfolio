Predicting Body Fat Using Linear Regression

Overview

This project uses linear regression to predict body fat percentage based on two key features: weight and abdomen circumference. The process included data visualization, outlier detection/removal, model training, and performance evaluation.

Problem Statement

Estimating body fat accurately is important in the health and fitness fields. Traditional methods can be invasive or expensive. This project explores a simple machine learning approach using basic physical measurements to make body fat predictions.

Tools Used

  • Python (Jupyter Notebook)
  • Pandas, NumPy
  • Matplotlib, Seaborn
  • Scikit-learn (Linear Regression)

Process

  1. Exploratory Data Analysis: Plotted weight and abdomen vs. body fat to understand patterns.
  2. Outlier Detection: Identified extreme outliers using IQR method and visual plots.
  3. Model Building: Used weight and abdomen as features (X) and body fat as the target (y).
  4. Model Evaluation: Compared performance before and after removing outliers using metrics like MAE, MSE, RMSE, and R².
  5. Visualization: Plotted Actual vs Predicted body fat to visualize model fit.

Model Performance

Before Outlier Removal:

  • MAE: 3.33
  • MSE: 17.98
  • RMSE: 4.24
  • R²: 0.613

After Outlier Removal:

  • MAE: 3.43
  • MSE: 18.36
  • RMSE: 4.28
  • R²: 0.605

Surprisingly, removing outliers slightly worsened the model performance. This highlights the importance of examining the effect of data cleaning on model outcomes.

Value

This project demonstrates practical skills in data cleaning, visualization, regression modeling, and performance interpretation. It also illustrates critical thinking about when cleaning may or may not help.

Reflection

I gained hands-on experience in handling real-world data challenges like noisy features and outliers. Even though model accuracy didn’t improve, I understood how model diagnostics and visual inspection are essential to trust predictions.
